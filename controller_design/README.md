# controller_design — Robust H∞ / Youla-H Power Balance Controller

Design workspace for the controller that replaced `PI_Controller_Power()` in
`teensy_controller/teensy_controller.ino`, following
*A Practical Youla Gain-Tuning Framework for H∞ Controller Realization*
(`references/A_Practical_Youla_Gain_Tuning_Framework_for_H__Controller_Realization_20260603.pdf`).

**Status (2026-07-10): designed, implemented, and host-validated.** The Youla-H
controller ships in the firmware (`USE_YOULA_SHARE_CONTROLLER = 1`; legacy PI retained
as a fallback path). Remaining work is bench calibration → regenerate (see the
recalibration loop in `controller_synthesis.md` §7).

## Key results

- Plant: $\alpha = r + \Delta V_0 r(1-r)/(k_d I_{tot})$ — exactly unity nominal gain,
  delay-dominated dynamics ($T_d \approx 1$ ms at the 1 kHz loop).
- H∞: $\gamma_{opt} = 0.653$; Youla-H shifts the Y gain +0.0036 % → $T(0) = 1$ exact.
- Implemented controller: integrator ($k_I = 111.9$) + 3 biquad states at $T_s = 1$ ms.
- Robustness: all 60 parameter corners stable; worst $M_2 = 0.59$; delay margin 11.7 ms.
- vs legacy PI at the worst corner (discrete $\|S\|_\infty$): **1.87 vs 26.9**.
